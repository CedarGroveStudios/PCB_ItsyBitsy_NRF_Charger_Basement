## Overview
A piggybottom LiPo charger board for attaching underneath an Adafruit ItsyBitsy NRF board. A 2-pin JST LiPo battery connection is on-board that supports charging (100mA charge rate), compatible with Adafruitâ€™s standard battery connector polarity.
